# The-Next-Thot

## Description

The Next Thot is a social media API that allows users to create accounts, post thoughts, and react to other users' thoughts. Users can also add/or friends to/from their friend list.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Questions](#questions)

## Installation

To install the necessary dependencies, run the following command:

    `npm i`

## Usage

To use the application, run the following command:

    `npm run start` or if you have nodemon installed, `npm run watch`
